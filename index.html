3 main differences between web1, web2, web3:


הבדל ראשון - חווית משתמש למול האינטרנט
web1 - אתרים סטטיים המאפשרים רק לצפות בתוכן ללא ביצוע פעילויות מעבר לצפייה
web 2 - נכנסה האפשרות לאינטרקטיביות למול המשתמש כמו פורומים ורשתות חברתיות המאפשרות לייצר תוכן
web 3 - מאפשר התאמה של התוכן למשתמשים ושימוש בכלים מתקדמים וטכנולוגיות כדי לשלוט על מידע ונתונים


הבדל שני - מי ייצר את האתרים?
web1 - בעיקר רשתות גדולות או ארגונים רשמיים על מנת לספק מידע
web 2 - נוספה האפשרות לפרט לייצר אתרים, בלוגים ורשתות חברתיות לשיתוף מידע פרטי
web 3 - ברמת השימוש די זהה ל2 אך השוני שמתנסה להתבצע הוא שלפרט תהיה יותר שליטה וכוח על המידע הפרטי


הבדל שלישי - חיפוש מידע
web1 - אין מנועי חיפוש המאפשרים לבצע חיפוש מידע ועיבוד שפה
web 2 - מאפשר להעלות שאילתות לאתרים ופורומים, אך לא היו את הכלים ולקבל תוכן הקשור לשאלה אך לא בהכרח את התשובה
web 3 - מאפשר חיפוש ע"פ כלים מתקדמים כמו בינה מלאכותית על מנת לאחזר את התשובות הרלוונטיות ביותר לשאילתות
